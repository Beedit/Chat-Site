# Chat Site
Some funny chat site im throwing together to see if i **can**

# Setup
## Prerequisites
- NodeJS
- npm
- git

## Setup
```sh
    git clone https://github.com/Beedit/Chat-Site
    cd Chat-Site
    npm i
    cp .env.example .env
    nano .env # set the port you want it to run on here
    npm run start
```
Should be running!
